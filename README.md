# homebrew-lector

Homebrew tap for [Lector](https://github.com/herbertwxin/Lector) — a keyboard-driven academic PDF viewer.

## Install

```bash
brew tap herbertwxin/lector
brew install --cask lector
```

## Update

```bash
brew upgrade --cask lector
```

Homebrew checks for updates automatically when you run `brew upgrade`. To enable scheduled background updates:

```bash
brew autoupdate start
```
